// 03. CARA DEPLOY

Host di
GitHub Pages.

1
Buat Repository Baru

Buka github.com/new. Beri nama repo, contoh: local-llm-setup. Set ke Public, lalu klik Create repository.

2
Siapkan Struktur Folder

Download semua file panduan ini, susun dengan struktur berikut sebelum upload:

📁 local-llm-setup/
│── index.html ← halaman ini
└── 📁 guides/
    ├── arc-b580.html
    └── universal.html
3
Upload via Git (Disarankan)

Clone repo, masukkan file, push. Atau upload langsung via Add file → Upload files di halaman repo GitHub.

4
Aktifkan GitHub Pages

Di repo → Settings → Pages → Source: Deploy from a branch → Branch: main, folder: / (root) → klik Save.

5
Tunggu 1–2 Menit

Setelah aktif, situsmu live di:
https://username.github.io/local-llm-setup

terminal — git deploy
