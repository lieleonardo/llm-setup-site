// Step 0 — Pilih GPU kamu

🟢 NVIDIA GeForce / RTX / GTX
🔴 AMD Radeon RX series
🔵 Intel Arc A-series / B-series

🟢 NVIDIA terdeteksi — menggunakan Ollama standar + CUDA. Cara paling mudah.
🔴 AMD Radeon — menggunakan Ollama standar + Vulkan (Windows). ROCm hanya di Linux.
🔵 Intel Arc — menggunakan IPEX-LLM (fork Ollama khusus Intel). Sedikit lebih banyak langkah.
  • Download driver terbaru di nvidia.com/drivers — pilih GPU GeForce / RTX / GTX kamu
  • Install dan restart PC
  • Verifikasi: buka CMD ketik nvidia-smi — harus muncul info GPU dan VRAM
CMD — verifikasi driver
nvidia-smi
# Output yang diharapkan:
+-----------------------------------------------------------------------------+
| NVIDIA-SMI 560.xx    Driver Version: 560.xx    CUDA Version: 12.x         |
| GPU  Name        VRAM        Driver-Model |
ℹ️CUDA tidak perlu diinstall manual — Ollama sudah bundling CUDA runtime-nya sendiri.
  • Download AMD Software: Adrenalin Edition terbaru
  • Install dan restart PC
⚠️ROCm di Windows hanya untuk GPU discrete Radeon RX (bukan APU/iGPU). Jika kamu pakai APU seperti Ryzen 7000G, gunakan mode Vulkan saja.
  • Download driver di Intel Download Center — cari "Intel Arc & Iris Xe Graphics"
  • Minimal versi 32.0.101.6078 atau lebih baru
  • Install Visual Studio 2022 Community — centang workload "Desktop development with C++"
  • Restart PC setelah install driver
✅NVIDIA adalah yang paling mudah — Ollama resmi sudah support CUDA out-of-the-box.
  • Download installer dari ollama.com/download/windows
  • Jalankan OllamaSetup.exe — install tanpa perlu admin rights
  • Verifikasi: buka CMD ketik ollama --version
CMD
# Cek versi Ollama
ollama --version

# Cek GPU terdeteksi (jalankan setelah ollama serve)
ollama ps
ℹ️Ollama standar sudah support AMD di Windows via Vulkan. Untuk performa lebih baik di GPU RX 6000+, bisa tambahkan ROCm.
  • Download OllamaSetup.exe dari ollama.com
  • Install seperti biasa
  • Aktifkan Vulkan backend dengan set environment variable:
Windows Settings → Environment Variables
# Tambahkan di System Environment Variables:
OLLAMA_VULKAN = 1

# Atau via PowerShell (permanen untuk user):
[Environment]::SetEnvironmentVariable("OLLAMA_VULKAN", "1", "User")
⚠️Jika GPU kamu RX 6000+ dan ingin ROCm (performa lebih tinggi), download juga dari ollama-for-amd dan replace file ROCm-nya.
⚠️Ollama standar tidak support Intel Arc. Harus pakai IPEX-LLM Portable ZIP dari Intel.
  • Buka github.com/intel/ipex-llm/releases
  • Download file ollama-ipex-llm-*.zip versi Windows terbaru
  • Ekstrak ke C:\ipex-llm
Struktur folder setelah ekstrak
C:\ipex-llm\
├── ollama.exe         # Binary Ollama versi Intel
├── start-ollama.bat   # Script jalankan server
└── ...
CMD — Terminal 1 (jangan ditutup)
# Jalankan server
ollama serve

# Output yang diharapkan:
Listening on 127.0.0.1:11434
✅Ollama otomatis mendeteksi GPU NVIDIA dan menggunakan CUDA. Tidak perlu set env variable tambahan.
CMD — Terminal 1
# Pastikan OLLAMA_VULKAN=1 sudah diset, lalu:
ollama serve

# Cek GPU terdeteksi di output:
inference compute  library=Vulkan  name=Vulkan0
description="AMD Radeon RX ..."
⛔Jika output menunjukkan library=cpu, pastikan OLLAMA_VULKAN=1 sudah tersimpan di Environment Variables dan restart terminal.
CMD — Terminal 1 (dari folder C:\ipex-llm)
:: Masuk ke folder IPEX-LLM
cd C:\ipex-llm

:: Set environment GPU Arc
set ONEAPI_DEVICE_SELECTOR=level_zero:0
set SYCL_CACHE_PERSISTENT=1
set OLLAMA_INTEL_GPU=true
set ZES_ENABLE_SYSMAN=1

:: Jalankan server
.\ollama.exe serve
✅Server berjalan jika muncul: Listening on 127.0.0.1:11434

Metode 1 — Per session (jalankan setiap kali):

    "DISABLE_PROMPT_CACHING": "1"

  "hasCompletedOnboarding": true,
  "primaryApiKey": "sk-dummy-key"
}
💡Untuk kembali ke Anthropic API: hapus env di settings.json atau jalankan set ANTHROPIC_BASE_URL= di terminal.
💡Open WebUI memberikan tampilan chat seperti ChatGPT, berjalan 100% lokal di browser kamu. Support semua GPU — NVIDIA, AMD, dan Intel Arc.

A. Requirement — pastikan sudah terinstall:

  • Python ≥ 3.11 — cek dengan python --version
  • Ollama sudah berjalan (selesaikan Step 1–4 terlebih dahulu)

B. Install Open WebUI via pip

CMD / PowerShell
# Install Open WebUI
pip install open-webui

C. Urutan menjalankan setiap kali pakai

CMD / PowerShell — terminal 1
# Jalankan Ollama terlebih dahulu
ollama serve
CMD / PowerShell — terminal 2
# Jalankan Open WebUI di terminal terpisah
open-webui serve

# Buka browser:
http://localhost:8080
⚠️Untuk Intel Arc: pastikan Ollama IPEX-LLM sudah running sebelum membuka Open WebUI. Open WebUI hanya sebagai frontend — inferensi tetap dilakukan oleh Ollama + IPEX-LLM di GPU kamu.

D. Setup pertama kali di browser

1
Buat akun admin
Halaman pertama langsung minta Sign Up — isi nama, email, dan password bebas. Akun pertama otomatis jadi Administrator. Semua data disimpan lokal, tidak dikirim ke mana pun.
2
Verifikasi koneksi Ollama
Masuk ke Admin Settings → Connections — pastikan Ollama URL terisi http://localhost:11434 dan statusnya Service Connection Verified. Biasanya sudah auto-detect.
3
Download model
Masuk ke Admin Settings → Models — ketik nama model di kolom Pull a model lalu klik Download. Model yang sudah di-pull lewat Ollama CLI sebelumnya otomatis muncul di sini.
4
Mulai chat
Klik New Chat di sidebar kiri, pilih model dari dropdown atas, dan mulai ngobrol. Bisa juga drag & drop dokumen/gambar ke chat box.
✅Setup selesai — Open WebUI siap dipakai. Untuk sesi berikutnya cukup jalankan ollama serve + open-webui serve lalu buka http://localhost:8080.
